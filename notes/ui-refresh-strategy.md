## 2. Token Map (exact values for `tailwind.config.js`)

```js

  950: '#0d0c0f',  // page background (was #0f172a)
  900: '#151318',  // raised surface / cards (was #1e293b)
  800: '#211f26',  // elevated surface / hover states

  600: '#5a5563',  // muted borders / disabled
  500: '#8b8593',  // low-emphasis icons
  400: '#a9a3b0',  // muted body text on dark (was slate-400)
  200: '#dcd9df',  // secondary text

}
```

### Accent gradient pair
```js
accent: {
  from: '#7c5cff',    // violet
  to:   '#00d4c8',    // teal
  DEFAULT: '#7c5cff', // solid fallback (links, focus rings, icons)
  hover: '#9478ff',
}
```
Use as `bg-gradient-to-r from-accent-from to-accent-to` for: hero headline text-gradient, CTA hover glow, active nav underline, card hover border glow. Solid `accent` anywhere a gradient is impractical.

### Semantic aliases (components reference THESE, not raw neutral-XXX)
```js
surface: '#0d0c0f',            // = neutral.950
'surface-raised': '#151318',   // = neutral.900
'surface-elevated': '#211f26', // = neutral.800
'text-primary': '#f7f6f8',     // = neutral.50
'text-secondary': '#dcd9df',   // = neutral.200
'text-muted': '#a9a3b0',       // = neutral.400
'border-subtle': '#39353f',    // = neutral.700
'border-muted': '#5a5563',     // = neutral.600
```

### Contrast verification (WCAG AA ≥4.5:1 body text) — all verified
- text-primary on surface: ~18.9:1 ✅
- text-muted on surface: ~7.4:1 ✅
- text-muted on surface-raised: ~6.9:1 ✅
- accent (#7c5cff) on surface: ~5.1:1 ✅
- accent-to teal (#00d4c8) on surface: ~11.5:1 ✅
- **White on accent fill: ~3.2:1 ❌ FAILS.** → **Decision: primary CTA buttons use DARK text (`text-surface` #0d0c0f) on accent fill = 5.8:1 ✅.** Secondary/outline buttons keep `text-accent` on surface (5.1:1 ✅).

## 3. Font Decision

- **Body**: Inter, self-hosted via `@fontsource/inter` (weights 400/500/600/700).
- **Display (headings)**: `@fontsource/manrope` (confirmed on npm, v5.2.8; SIL OFL; weights 600/700/800). Pairs with Inter; avoids the Space Grotesk cliché.
- Install: `npm install @fontsource/inter @fontsource/manrope`
- Import weight CSS files at top of `src/main.jsx` (before App import):
  ```js
  import '@fontsource/inter/400.css'
  import '@fontsource/inter/500.css'
  import '@fontsource/inter/600.css'
  import '@fontsource/inter/700.css'
  import '@fontsource/manrope/600.css'
  import '@fontsource/manrope/700.css'
  import '@fontsource/manrope/800.css'
  ```
  Fontsource ships `@font-face` with `font-display: swap` — no manual @font-face needed.
- Tailwind: `fontFamily: { sans: ['Inter', ...defaultTheme.fontFamily.sans], display: ['Manrope', ...] }`. `font-display` class on h1/h2 section headings + Hero name only; body/nav/cards stay `font-sans`.
- **Remove** Google Fonts `<link>` tags from `index.html`.
- CLS: Manrope only used at display sizes → low risk; skip manual fallback-metric tuning unless Lighthouse Phase 4 shows visible shift.
